# Performa Layanan Penunjang - Frontend

Frontend application untuk sistem penilaian performa layanan penunjang rumah sakit.

## 🚀 Tech Stack

- **React 18** - UI Library
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Zustand** - State Management
- **React Router** - Routing
- **Axios** - HTTP Client
- **Framer Motion** - Animations
- **HTML5 QR Code** - QR Scanner
- **React Hot Toast** - Notifications

## 📦 Installation

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env
